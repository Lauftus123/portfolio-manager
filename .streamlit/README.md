# InvestIQ - AI Portfolio Manager (Streamlit)

## Setup

1. Clone this repo
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run locally:
   ```bash
   streamlit run app.py
   ```

## Deploy to Streamlit Cloud

1. Push this repo to GitHub
2. Go to https://share.streamlit.io
3. Connect your GitHub account
4. Select this repo → `app.py`
5. Click Deploy!

## Usage

- **Dashboard** — portfolio overview & charts
- **Portfolio** — manage holdings (add/remove)
- **Transactions** — log buy/sell activity
- **AI Advisor** — chat with AI (requires OpenAI API key)
